### Requirements
* Ubuntu 16.04+
* NodeJS 8+
* Truffle v4
* MetaMask Plugin
* Ganache CLI

### Building
```sh
$ npm install
$ ganache-cli &
$ truffle migrate --network integration
```

### Running Tests
```sh
$ truffle test --network integration
```

### Running Web App

```sh
$ npm start
```
